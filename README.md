# tmux

My tmux setup (originally taken from [Omarchy tmux bindings](https://learn.omacom.io/2/the-omarchy-manual/53/hotkeys#tmux))

https://github.com/tmux/tmux/wiki

Copy the contents of this repository to `~/.config/tmux`.
The following files can be ignored:

- `README.md`
- `.git`
- `.gitignore`

## macOS Homebrew

Install the following packages:

```shell
brew install tmux
```

## ArchLinux / Omarchy

Install the following packages:

```shell
yay -S tmux
```
